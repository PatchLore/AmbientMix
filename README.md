# AmbientMix

AmbientMix is a browser-based ambient audio mixer that blends your music with rain, thunder, wind and other ambience layers. Export extended ambient loops directly in the browser using the Web Audio API.

## Features
- Upload MP3/WAV
- Layer multiple ambience FX
- Adjust volumes and mix levels
- Auto-loop for 10â€“60+ minutes
- Export high-quality WebM audio
- Fully client-side (no backend needed)

## Tech Stack
Next.js 15, TypeScript, Tailwind, shadcn/ui, Web Audio API, MediaRecorder
